# Manual Email Fixtures

This folder is for manually copied full HTML emails used by
`pnpm send:email:preview --source=file`.

Expected optional files:

```txt
manual-emails/
  full-flight-established.html
  full-flight-sales.html
  full-flight-insurance.html
```

These files are diagnostic fixtures only. They are not generated by
`pnpm build:emails` and are not part of the official dist output contract.
